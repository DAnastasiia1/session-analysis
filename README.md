# 📊 Web Session Analysis (Tableau Project)

## 🔍 Business Context

The goal of this project was to analyze website traffic and understand user behavior to support marketing and product decisions.

## 🛠 Tools
	•	Tableau (data visualization, dashboards)

## 📈 What I did
	•	Built an interactive dashboard to monitor web sessions
	•	Analyzed traffic sources (Direct, Organic, Paid, Social)
	•	Identified top-performing countries
	•	Evaluated device usage (desktop, mobile, tablet)
	•	Tracked session trends over time

## 💡 Key Insights
	•	Organic Search and Direct channels drive the majority of traffic
	•	Desktop users dominate, but mobile usage is growing
	•	A small number of countries generate most of the traffic
	•	Traffic fluctuates significantly over time

## 🎯 Business Value
	•	Helps marketing teams optimize channel performance
	•	Identifies high-value regions for targeting
	•	Supports decisions on mobile vs desktop optimization
	•	Enables monitoring of traffic trends and anomalies
